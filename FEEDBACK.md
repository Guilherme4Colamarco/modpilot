# 💬 Feedback & Sugestões — GamePiLot

Obrigado por usar o **GamePiLot**! Este arquivo é o espaço da comunidade para registrar feedbacks, reportar bugs e sugerir novas funcionalidades.

> **Nota:** Para feedbacks mais detalhados ou discussões, abra uma [Issue no GitHub](../../issues) ou envie um Pull Request.

---

## 🐛 Como Reportar um Bug

Ao reportar um problema, inclua:

- **Descrição:** O que aconteceu e o que era esperado
- **Passos para reproduzir:** Como replicar o problema
- **Ambiente:** Distro Linux, versão do Wine/Proton, launcher usado (Steam/Heroic/etc)
- **Logs:** Cole o log do terminal ou da interface do GamePiLot

### Exemplo de reporte:

```
**Bug:** O popup de erro aparece mesmo quando a instalação foi bem-sucedida.

**Passos:**
1. Selecionar Skyrim AE no Heroic GOG
2. Selecionar SKSE64 nas dependências
3. Clicar em "Instalar"

**Ambiente:** Arch Linux, proton-cachyos-slr, Heroic 2.x

**Log:**
⚠️ Instalador retornou erro: winediag: this wine contains many experimental patches...
```
